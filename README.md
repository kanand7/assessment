# Money APP

## npm

```npm install``` - Install all the dependencies. 

```npm run dev``` - Start the application locally `localhost:8080`

## Commands
```npm run build-prod``` - builds production build. This build does not include sourcemaps.  
```npm run dev``` - runs a server to track changes of project files and automatically builds development build.  
```npm run eslint``` - shows all linting problems.  
```npm run eslint-fix``` - fixes linting problems where possible.
